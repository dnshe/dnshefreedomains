## 📖 平台简介
**DNSHE** ([www.dnshe.com](https://www.dnshe.com)) 是一个高性能域名分发平台。我们的使命是为全球开发者、学生和开源爱好者降低互联网创作门槛，提供基础设施级的 DNS 服务。

通过采用企业级的 Anycast 技术，我们为您的项目提供稳定、快速的解析服务——且完全免费。

---

## ✨ 为什么选择 DNSHE？

### 📅 行业领先的续期政策 (180 天)
不同于其他临期才能续期的平台，DNSHE 开放了长达 **180 天（6个月）** 的续期窗口。您可以有充足的时间通过手动或自动化脚本维护域名，确保服务永不意外下线。

### 💎 企业级 Anycast 架构
- **低延迟**：全球范围内毫秒级的解析响应速度。
- **高可用性**：多冗余节点确保 99.9% 的解析正常运行时间。
- **全协议支持**：标准支持 A, AAAA, CNAME, MX, TXT, NS, SRV 和 CAA 记录。

### 🤖 自动化管理与 API 优先
DNSHE 专为自动化而生。您可以轻松将域名管理集成到 CI/CD 流水线或自动化 SSL 证书申请中。

## 🚀 快速开始

1.  **注册账号**：在 [dnshe.com](https://www.dnshe.com) 注册，无需信用卡。
2.  **申请域名**：搜索并申请您心仪的后缀。
3.  **管理记录**：添加服务器 IP，解析秒级全球同步。
4.  **维护续期**：在 180 天续期窗口内通过控制台或 API 完成续期。
